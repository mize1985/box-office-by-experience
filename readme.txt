*Data is not included because of file size limitations*

WebScraper.ipynb is a Jupyter notebook that scrapes movie data from Box Office Mojo and saves it into pickled dictionaries.

Regression.ipynb is a Jupyter notebook that contains cleaning, EDA, and modeling of the data, including producing numerous plots.

Presentation.pdf contains all the slides to be used in the presentation to the class. Results of data analysis and modeling performed in Regression.ipynb may be found there.

And the media folder contains all images used in the slideshow and a few more.
